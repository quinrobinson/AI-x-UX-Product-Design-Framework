# APDF Inputs

Drop raw project material here before running autonomous Orchestrator commands.

| File | Used by | Contents |
|------|---------|----------|
| `session-notes.md` | `/discover` | Interview transcripts or research session notes |
| `competitor-list.md` | `/discover` | Products to analyse — names, URLs, brief descriptions |
| `service-scope.md` | `/discover` | Optional: persona, goal, and service channels for blueprint |
| `screen-inventory.md` | `/deliver` | List of all screens to be handed off to engineering |
| `qa-notes.md` | `/deliver` | Optional: raw QA observations from implementation review |

Files here are read by the Orchestrator but not auto-generated.
Outputs are written to `.apdf/artifacts/` automatically.
